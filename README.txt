The website, Le perfume is a perfume themed website that is designed to offer aesthetically pleasing and an interactive experience for the user.
In the website, the users are able to explore perfume options thriugh an image slider.
To build the website I have used HTML, CSS, JAVASCRIPT and XML

Features:
1. Interactive Image Slider: The website features an interactive image slider that allows users to explore perfume options
2. Aesthetically Pleasing Design: The website has an aesthetically pleasing design that is appealing and attractive and is also gentle for the user
3. Easy Navigation: The website has easy navigation that allows users to easily find and explore perfume options
